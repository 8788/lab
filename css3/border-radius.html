border-radius

语法

border-radius ： none | {1,4} [/ {1,4} ]?
相关属性：border-top-left-radius, border-top-right-radius, border-botton-left-radius, border-bottom-right-radius
统一设置各边圆角

普通圆角水平半径与垂直半径相等

.border-radius-1{
	width:100px;
	height:100px;
	background-color:pink;
	-moz-border-radius:10px;
	-webkit-border-radius:10px;
	border-radius:10px;
}

特殊圆角水平半径与垂直半径不相等

.border-radius-2{
	width:100px;
	height:100px;
	background-color:pink;
	-moz-border-radius:10px/30px;	/*==="/"前表示水平半径，"/"后表示垂直半径===*/
	-webkit-border-radius:10px/30px;
	border-radius:10px/30px;
}
单独设置某边圆角以左上角为例

普通圆角水平半径与垂直半径相等

.border-radius-3{
	width:100px;
	height:100px;
	background-color:pink;
	-moz-border-top-left-radius:10px;
	-webkit-border-top-left-radius:10px;
	border-top-left-radius:10px;
}

特殊圆角水平半径与垂直半径不相等

.border-radius-4{
	width:100px;
	height:100px;
	background-color:pink;
	-moz-border-top-left-radius:10px 30px; /*=== 注：此处水平半径与垂直半径之间无 "/" ===*/
	-webkit-border-top-left-radius:10px 30px;
	border-radius:10px 30px;
}
说明：border-top-right-radius, border-bottom-left-radius, border-bottom-right-radius属性同border-top-left-radius
