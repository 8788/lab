text-shadow

语法

text-shadow：none | <shadow> [ , <shadow> ]*
<shadow> = <length>{2,3} && <color>?

说明

值为none时表示无阴影
<shadow>中可取2-3个长度值和一个颜色值
1.第1个长度值用来设置对象的阴影水平偏移值。可以为负值
2.第2个长度值用来设置对象的阴影垂直偏移值。可以为负值
3.第3个值可选，如果提供了第3个长度值则用来设置对象的阴影模糊值。不允许负值 4.最后一个值为阴影颜色，也可用rgba或hsla的透明色

举例

我的前端实验室

.text-shadow-1{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:0 0 10px #000; /*阴影无偏移*/
}

我的前端实验室

.text-shadow-2{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:3px 3px 5px #f00; /*阴影偏移3px 3px*/
}

我的前端实验室

.text-shadow-3{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:3px 3px 5px #f00; /*当然偏移也可以是负值*/
}

利用rgba实现阴影透明

我的前端实验室

.text-shadow-4{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:0 0 5px rgba(0,0,0,.5);
}

我的前端实验室

.text-shadow-5{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:3px 3px 5px rgba(0,255,255,.3);
}

还可以通过设置多个阴影来实现一些特效

浮雕效果

.text-shadow-6{ 
    padding-left:30px; 
    font:18px/24px "Microsoft YaHei"; 
    text-shadow:-1px -1px 0 #fff, 1px 1px 0 #666;
}

3D效果

.text-shadow-7{ 
    padding:10px 30px; 
    font:bold 30px/40px "Microsoft YaHei"; 
    background-color:#999; 
    color:#fff; 
    text-shadow: 1px 1px rgba(197, 223, 248,0.5),2px 2px rgba(197, 223, 248,0.5),3px 3px rgba(197, 223, 248,0.5),
    4px 4px rgba(197, 223, 248,0.5),5px 5px rgba(197, 223, 248,0.5),6px 6px rgba(197, 223, 248,0.5);}

描边效果

.text-shadow-8{ 
    padding-left:30px; 
    font:bold 30px/40px "Microsoft YaHei"; 
    background-color:#999; color:#fff; 
    text-shadow:-1px -1px 0 #f00; 1px 1px 0 #f00;
}
